Candy Simulation Task (C++)
Цей проєкт я створив під час навчання в межах вивчення базових алгоритмів та логічних циклів.

Технології:
C++

Бібліотека <windows.h> — використана для коректного відображення кирилиці в консолі.

Що робить ця програма:
Моделює споживання ресурсів: Програма імітує покроковий процес, де два персонажі (Сем та Юра) одночасно "з'їдають" по одній одиниці ресурсу.

Візуалізує кожен крок: Консоль виводить детальний лог кожного кроку з актуальним залишком для кожного учасника.

Обчислює фінальний результат: Після завершення циклу (коли у когось закінчуються ресурси) програма видає загальну кількість цукерок, що залишилися.
